# BITACORA TERRENO

Aplicacion movil HTML de registro en terreno asociada a `ZALINEL SUITE`.

## Fuente actual

La version operativa vigente se toma desde:

- `ZAL-TOOL-BITACORA-TERRENO-V5_1.html`

## Criterio de versionado

- este repositorio debe guardar la app vigente y su documentacion minima
- los respaldos locales se conservan fuera de Git
- la carpeta `CODEX/` puede mantener notas de trabajo, pero no sus backups historicos

## Relacion con la Suite

`BITACORA TERRENO` sigue siendo una aplicacion separada, pero forma parte del ecosistema operativo de ZALINEL y debe mantenerse coherente con la arquitectura de `ZALINEL SUITE`
